Brief Explanation about HTML

What is Html and how it is work

How Html works

HTML (HyperText Markup Language) is the standard language used to create and design web pages. It provides the structure and content of a webpage, allowing developers to define headings, paragraphs, images, links, tables, and more.

Html works in a nutshell:

  • HTML works by using tags and attributes. Tags are enclosed in (<>), and most tags have a closing tag
  • When you open a webpage, the browser (e.g., Chrome, Firefox) reads the HTML code, interprets it, and displays the content in a visually structured way. It doesnâ€™t show the raw code but processes it to render the layout.
  • Tags define the roles of different elements, such as headings, paragraphs, images, and links

HTML Structure

Every HTML document follows a standard structure, ensuring browsers can interpret and render it correctly. Below is the basic anatomy of an HTML document:

            <!DOCTYPE html>
            <html lang="en">
              <head>
                <title>My Page</title>
              </head>
              <body>
                <h1>Welcome</h1>
                <p>This is a paragraph.</p>
              </body>
            </html>

History about Html

  1. 1989:The birth of Html
    • Tim Berners-Lee, a physicist at CERN, proposed a system to share and link documents across the internet.
    • This proposal led to the creation of HTML (HyperText Markup Language) as part of the World Wide Web.
  2. 1991:First Html Version
    • Tim Berners-Lee published the first description of HTML.
    • It included 18 tags, many of which are still in use today, like <title>, <p>, <h1>, <a>, etc.
  3. 1995:Html 2.0
    • The Internet Engineering Task Force (IETF) created the first official HTML standard.
    • HTML 2.0 introduced basic forms and tables, setting a foundation for interactive websites.
  4. 1997:Html 3.2
    • Managed by the World Wide Web Consortium (W3C), HTML 3.2 became the official standard.
    • It introduced features like JavaScript support, applets, and more complex table designs
  5. 1999:Html 4.01
    • A major update, providing better support for multimedia and accessibility.
    • Introduced <div>, <span>, and improved forms.
  6. 2008:Html 5 Draft
    • With the rise of multimedia and interactive websites, the WHATWG (Web Hypertext Application Technology Working Group) and W3C collaborated on HTML5.
    • It addressed the needs of modern web development, focusing on native video, audio, and semantic elements like <header>, <footer> , <article> , etc.
  7. 2014:Html 5 official release
    • HTML5 became the official standard.
    • It removed the dependency on plugins (like Flash) and introduced APIs for offline storage, geolocation, and more.
