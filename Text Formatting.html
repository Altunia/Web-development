Explanation of Text Formatting

Now, let understand some text Formatting text in html one by one like:

  • <b>

    • This text is bold


    Bold and Strong Text: Used to emphasize text with greater weight, typically making it bold

  • <i>

    • This is Itlic form


  • <emp>

    • The text is Emphasized

    Italic and Emphasized Text: Used to slant the text or provide emphasis with a different styling.

  • <u>

    • It is used for underlined text or word


    Underlined Text: Highlights text with an underline.

  • <sub>

    Formula of water is H2O

  • <sup>

    • formula of nitrate is N 3O


    Subscript and Superscript: Adjusts the positioning of text, placing it below or above the normal line of text.

  • <del>

    • The text is deleted

    The del tag: is used to indicate text that has been deleted or is no longer relevant.

  • <mark>

    • The text is marked

    Highlighted Text: Marks text with a background color for emphasis.

  • <small>

    • This is text small

    Small Text: Reduces the size of the text relative to the default font size.

  • <q>

    • This is short line quotation

  • <blockquote>

    • This is long block quotation

    Inline and Block Quotes: Represents quoted text, either inline within a paragraph or as a separate block element.

  • <pre> 

    •             Abid Ullah
            Asim
            Fawad

    Preformatted Text: Maintains whitespace and line breaks, displaying content exactly as written in the source code.
